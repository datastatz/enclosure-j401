# V2 Enclosure Dimensions

## Orientation

The **I/O side** (open side) is the reference point and is considered the **bottom/front**. All directions are relative to this side:

- **Left** = left when facing the I/O side
- **Right** = right when facing the I/O side
- **Top** = the side directly opposite the I/O side
- **Bottom** = the underside of the enclosure (where the fan is)
